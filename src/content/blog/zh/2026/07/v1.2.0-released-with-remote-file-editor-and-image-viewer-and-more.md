## ✨ v1.2.0 更新内容

我们很高兴地宣布 SSH Config Manager v1.2.0 正式发布！本次更新对远程文件管理体验进行了大幅升级，同时对整个应用的视觉风格进行了全面重构。

### 📁 文件管理器重构：新增目录树

文件管理器经历了彻底的结构重构。此前仅支持扁平的文件列表展示，现在升级为**左侧目录树 + 右侧文件列表**的双栏布局。这种经典的 IDE 风格导航方式，让浏览深层目录结构变得直观高效。

![file manager](@/assets/screenshots/zh/file_manager-explorer.png)

### 🎨 全局视觉风格重构

我们对整个应用的视觉风格进行了全面改版。主页、登录页和文件管理页面原有的卡片式设计已被移除，取而代之的是**更专业、更素雅的扁平化设计**。新的统一视觉语言减少了视觉干扰，提供了更专注、更优雅的用户体验，与现代开发者工具的审美标准保持一致。

![main page](@/assets/screenshots/zh/main_page.png)

### ✏️ 内置远程代码编辑器

现在可以直接在浏览器中编辑远程文件，无需先下载到本地。内置的代码编辑器支持：
- 数十种编程语言的语法高亮
- 多行编辑与编码切换（UTF-8/GBK）
- 换行符格式检测与转换（LF/CRLF）
- 外部修改冲突检测

![remote file/code editor](@/assets/screenshots/zh/file_manager-editor.png)

### 🖼️ 内置图片查看器

无需下载即可即时预览远程服务器上的图片。集成的图片查看器支持常见格式（PNG、JPG、GIF、WebP、BMP 等），并支持缩放和平移操作，方便查看细节。

![remote image viewer](@/assets/screenshots/zh/file_manager-image-viewer.png)


### 🗂️ 全新文件图标系统

文件和文件夹现在会根据类型显示**独特的彩色图标**。编程语言、配置文件、图片、压缩包和文档各有专属的视觉标识，让您在浏览目录时一目了然。

### 🪟 自定义模态对话框

原生的浏览器 `confirm()` 和 `prompt()` 弹窗已被替换为**精心设计的自定义模态窗口**，用于删除确认、重命名操作以及新建文件/文件夹。这些模态框提供了一致的视觉体验，支持键盘快捷键（回车确认、ESC 取消），并彻底消除了浏览器"阻止此页面创建更多对话框"的烦人提示。

### 🔧 其他改进

- 文件编辑器标签页支持拖拽排序
- 右键菜单新增智能"新建文件"和"新建文件夹"选项
- 后端新增全操作审计日志
- 改进的错误处理与连接稳定性
