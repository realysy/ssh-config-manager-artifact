这个工具的基本功能已经完成：

![主页](@/assets/screenshots/main_page.png)

- 支持分组管理 SSH 连接。
- 分组文件格式与 `~/.ssh/config` 完全一致，这意味着你可以直接用 `ssh` 命令连接服务器。
- 如果你用这个工具编辑 `~/.ssh/config` 中的配置，所有字段和自定义注释都会被保留，即使它们不被本软件支持。
- 默认分组使用 `/etc/ssh/ssh_config` 作为默认配置文件。
- 自定义分组文件将存储在自定义目录中。
- 可以测试 SSH 连接的连通性。
- 可以通过 `wezterm` 连接 SSH，wezterm 是一个终端模拟器，需要预先安装在你的系统上。
- 以上所有功能均已测试通过。

- 一个简单的认证系统。
