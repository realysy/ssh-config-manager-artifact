🔧 SSH Config Manager

This is a tool to manage SSH connections in groups, using the same format as ~/.ssh/config. It supports connectivity testing and launching connections via:

🖥️ WezTerm 🖥️ Alacritty 🖥️ Kitty 🪟 Windows Terminal 🐧 GNOME Terminal 🐧 KDE Konsole

💡 Flexible Configuration

The config file of custom groups is stored in a self-defined folder. You can also use it manually like:

ssh -F ~/my_ssh_config/test_group.config my_host

...or use in other tools which support SSH config format.
